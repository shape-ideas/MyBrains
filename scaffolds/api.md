## 示例

### 请求

```bash
curl ${BaseURL}/${API_PATH} \
    -H "Content-Type: application/json;charset=UTF-8" \
    -d '{
          "key": "val"
        }'
```


### 返回


```json
    {
      "data": {
        "key": "val"
      }, 
      "status": {
        "code": "E0", 
        "message": "OK"
      }
    }
```
